引导问题

为 OH 卡教练设计：按「提问类型」整理示例问题，帮助你在带领时学会如何发问、何时深化，让探索更安全、更有效。

  • 你看到了什么？

  • 画面里有什么？有哪些细节吸引你？

  • 这张卡里有什么你从未注意过的细节？

  • 给这张卡起一个名字。

  • 第一眼你注意到的是什么？

  • 画面中的色彩、线条或形状，有什么让你有感觉？

  • 如果这张图会动，你觉得会怎么动？

  • 画面里有没有让你想靠近或想远离的部分？

  • 这张卡和最近的你有什么联系？

  • 画面中你在哪里？在做什么？

  • 这张卡代表你现在的情绪状态是什么？

    → 单卡直觉牌阵

  • 这张卡如何反映「你是谁」？

    → 单卡直觉牌阵

  • 此刻的生活里，有什么和这张卡很像？

  • 这张卡像你生命中的哪一段或哪一个人？

  • 如果这张卡是今天的你，它在说什么？

  • 你现在的状态，用这张卡来比喻会是什么？

  • 你的情绪是如何从过去发展到现在，又将走向哪里？

    → 过去·现在·未来

  • 这段关系是如何发展的？未来会如何？

    → 过去·现在·未来

  • 过去、现在、未来三张卡，如何照亮你的选择？

    → 过去·现在·未来

  • 你的生涯过去、现在与理想未来，分别是什么画面？

    → 过去·现在·未来

  • 如果过去是这张卡，现在和未来可能是什么样子？

  • 一年前的你看到这张卡，和现在的你看到，会有什么不同？

  • 这件事如果继续发展下去，你希望下一张卡是什么画面？

  • 此刻，你身体里的压力在哪里？

  • 你的身体现在需要什么？

  • 在什么情况下你觉得界限被越过了？感觉如何？

  • 若愿意，这张卡代表的那段经历在身体哪里？

    → 创伤见证牌阵

  • 看着这张卡，你身体哪个部位有反应？

  • 这种感受如果有一个温度或重量，会是什么？

  • 你的呼吸现在是什么样的？

  • 肩膀、胸口或肚子，哪里最想被注意到？

  • 愤怒背后，你真正需要的是什么？

  • 这股愤怒在保护你什么？

  • 这张卡与你的焦虑或恐惧有什么联系？

    → 单卡直觉牌阵

  • 你允许自己拥有这份情绪吗？

  • 失去之后，什么还留着？

  • 带着这份失落，你还能相信什么？

  • 什么让你感到滋养与充电？

  • 什么能帮助你在压力中「充电」？

  • 今天的你，比当时的你多了什么力量？

  • 谁或什么曾经给过你类似的支持？

  • 在这张卡里，有没有一点光或一点温暖？

  • 你曾经克服过什么，是当时的你没想到的？

  • 有什么是你一直想对自己说的？

  • 如果排除所有人的期待，你真正想选的是哪个？

  • 哪个选择更靠近「你想成为的样子」？

  • 你的身体对哪个选项更有反应？

  • 如果五年后回头看，你希望自己当时怎么选？

  • 从今天的探索里，你带走的一个小小的行动或尝试会是什么？

  • 如果只能做一件事，你最想先做哪一件？

  • 接下来的一周，你愿意为自己做的一件小事是什么？

  • 这张卡提醒你要如何照顾自己？

  • 明天或接下来几天，你想从哪里开始？

  • 做什么会让你离「你想要的」更近一点？

  • 你愿意让谁知道你的这个小决定？

  • 这段关系里，你在重复谁和谁的模式？

  • 你希望对方怎么对待你？你愿意怎么对待自己？

  • 如果关系可以有一个小小的改变，你希望是哪一点？
